# Muraena Configuration – Educational & Research Use Only

## ⚠️ Disclaimer

This repository contains configuration files and experiments related to the use of the **Muraena reverse proxy framework**.

**This project is strictly for educational and research purposes only.**

It is intended to help understand:

* How reverse proxy frameworks operate
* How modern web authentication flows behave
* How credential interception techniques work (from a defensive perspective)
* How organizations can detect and mitigate such threats
